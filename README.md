# рҹӣ’ Mart POS System

A modern **Point of Sale (POS)** бһҹбһҳбҹҗбһҷбһ‘бҹҶбһ“бһҫбһ” бһҠбҹӮбһӣбһ”бһ¶бһ“бһҡбһ…бһ“бһ¶бһЎбһҫбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһӣбһҖбҹӢбһҡбһ¶бһҷбһұбҹ’бһҷбһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бҹ” бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ“бҹҒбҹҮбһҮбһҪбһҷбһұбҹ’бһҷбһўбһ¶бһҮбһёбһңбһҖбһҳбҹ’бһҳбһўбһ¶бһ…бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһ¶бһҡбһӣбһҖбҹӢ бһ•бһӣбһ·бһҸбһ•бһӣ бһўбһҸбһ·бһҗбһ·бһҮбһ“ бһ“бһ·бһ„бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһ•бҹ’бһҹбҹҒбһ„бҹ—бһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣ бһҮбһ¶бһҳбһҪбһҷбһ“бһ№бһ„бһ•бҹ’бһ‘бҹғбһҳбһ»бһҒбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ (UI) бһҠбҹӮбһӣбһҹбҹ’бһўбһ¶бһҸ бһ“бһ·бһ„бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһ”бҹ’бһҡбһҫбҹ”
